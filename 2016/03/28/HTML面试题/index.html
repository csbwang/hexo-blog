HTML面试题

发表于 2016-03-28   |   分类于 Web   |  
  1. Doctype作用? 严格模式与混杂模式如何区分？它们有何意义?
  • <!DOCTYPE>声明位于文档的最前面，告知浏览器的解析器使用什么文档类型规范来解析当前文档
  • 严格模式下的排版和js运作模式都是以当前浏览器的最高标准执行
  • 混杂模式时页面以宽松的向后兼容的方式显示和运行js，模拟老式浏览器的行为以防止站点无法工作
  • <!DOCTYPE>不存在或者格式不正确会导致浏览器使用混杂模式
  1. 行内元素、块级元素、空元素有哪些？
  • 行内元素：span a input image b select strong
  • 块级元素：div ul ol dl dt dd p h1~h6
  • 空元素：开始标签和结束标签之间没有内容的元素称为空元素，有input image br hr link meta等
  1. link和@import的区别
  • link属于XHTML标签，@import是CSS提供的
  • 页面被加载时，link会同时被加载，而@import引用的CSS会等到页面被加载完再加载
  • @import只有IE5以上版本才能识别，link属于XHTML标签，没有兼容问题
  • link方式的样式的权重高于@import的权重
  • link属于XHTML标签，除了加载CSS外，还能用于定义RSS，定义rel连接属性等，而@import只能用于加载CSS
  1. 浏览器的内核分别是什么？
  • IE的浏览器内核Trident（360 搜狗 遨游浏览器等也用）
  • Mozilla的Gecko
  • Chrome的Blink（WebKit的分支）
  • Opera的内核原为Presto，现为Blink
  • Safari的内核为WebKit
  1. 介绍下你对浏览器内核的理解
  • 浏览器内核主要分两部分：渲染引擎和JS引擎
  • 渲染要引擎主要负责获取页面的内容，整理样式以及计算网页的显示方式
  • 解析和执行javascript代码来实现网页的动态效果
  1. xhtml和html有什么区别
    HTML是一种基本的WEB网页设计语言，XHTML是一个基于XML的置标语言，最主要的不同：
  • XHTML 元素必须被正确地嵌套。
  • XHTML 元素必须被关闭。
  • 标签名必须用小写字母。
  • XHTML 文档必须拥有根元素。
  1. 前端页面有哪三层构成，分别是什么?作用是什么?
  • 结构层 HTML
  • 表示层 CSS
  • 行为层 JavaScript
  1. img标签上title与alt属性的区别是什么?
  • alt：当图片不显示时用文字代表。
  • title 表示鼠标在图片上停留时,显示一个悬浮框,其中显示的文字。
