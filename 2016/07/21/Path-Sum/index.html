Path Sum

发表于 2016-07-21   |   分类于 Leetcode   |  

题目要求：判断一颗二叉树从是否存在一条从根节点到叶子节点的路径，他们的节点值之和等于给定的整数
思路：递归遍历这颗二叉树，如果只有一个节点，看其值是否等于给定值，否则递归判断左右子树
