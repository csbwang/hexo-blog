Binary Tree Paths

发表于 2016-07-25   |   分类于 Leetcode   |  

题目要求：给定一棵二叉树，求所有从根节点到叶子节点的路径
思路：

  1. 采用递归，当前节点到也叶子节点的路径包括

    • 当前节点到左子树根节点的路径加上左子树根节点到所有叶子节点的路径
    • 当前节点到右子树根节点的路径加上右子树根节点到所有叶子节点的路径

  2. 深度遍历二叉树，遍历的过程中记录之前的路径，一旦遍历到叶子节点便将该路径加入结果中
