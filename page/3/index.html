HTML面试题

发表于 2016-03-28   |   分类于 Web   |  
  1. Doctype作用? 严格模式与混杂模式如何区分？它们有何意义?
  • <!DOCTYPE>声明位于文档的最前面，告知浏览器的解析器使用什么文档类型规范来解析当前文档
  • 严格模式下的排版和js运作模式都是以当前浏览器的最高标准执行
  • 混杂模式时页面以宽松的向后兼容的方式显示和运行js，模拟老式浏览器的行为以防止站点无法工作
  • <!DOCTYPE>不存在或者格式不正确会导致浏览器使用混杂模式
